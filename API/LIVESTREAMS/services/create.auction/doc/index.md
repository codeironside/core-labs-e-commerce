# Create Auction — Service Documentation

## Overview

Opens a livestream auction for a product already listed on the vendor's livestream.

## Endpoint

| Property | Value |
|---|---|
| Method | `POST` |
| Path | `/api/v1/livestreams/vendor/:livestreamId/auctions` |
| Content-Type | `application/json` |
| Auth Required | ✅ |
| Required Role | `vendor` |
| Tags | Livestreams — Vendor |
